Lab 01

Overview

This lab introduced creating and deploying a website using GitHub and SSH.

Using SSH

Securely connect to your server using:

ssh yourDomainName

Using cd

Navigate directories using:

cd /var/www/html/iit/

Pulling from GitHub

Update your live site using:

sudo -u www-data git pull
