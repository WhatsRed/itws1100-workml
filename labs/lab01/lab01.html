Information/Notes

How to use "ssh" (Secure Shell)

Using your command prompt, you must ssh into your website using the command:

ssh [yourDomainName]

How to use "cd"

You can use "cd" to navigate to folders within your file explorer and change them

The command looks something like: cd /var/www/html/iit/

With the text after "cd" being the pathway to your file/folder

How to pull from GitHub

After committing your items to your repo, you need to pull your code to your website in order for your website to change

You can use the command "sudo" (Super User Do) in order to bypass authorization

Follow this with "-u www-data", which runs the command as the user www-data, which is the default web server on many linux systems

Finally followed by "git pull", which pulls from your github after you have committed

The entire command should look something like this:

sudo -u www-data git pull

Note: You must be cd'd into your file directory for this to work
