Basic Page Structure

<div class="container">
        <h2>Lab 02</h2>

This structure organizes content using semantic HTML and reusable classes. The container keeps everything centered and responsive.

Flexbox Layout (from CSS)

.navbar .container {
        display: flex;
        justify-content: space-between;
        align-items: center;


        }

Flexbox is used to align elements horizontally and evenly distribute space. This is how the navigation bar is structured cleanly.

Card Component

.card {
        background: white;
        padding: 25px;
        border-radius: 10px;
   

        }

The card system creates reusable content blocks that improve readability and give the site a modern design.
